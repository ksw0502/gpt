수어를 활용한 ChatGPT



수어는 소리가 아닌 손짓을 이용해서 뜻을 전달하는 언어입니다.

딥러닝을 통한 수어 인식 인공지능을 통해 수어를 인식하고 Chat GPT와 대화하는 서비스를 제공합니다.


서비스 사용하기



Text To ChatGPT


Open AI사에서 제공하는 API를 이용하여 Chat GPT를 사용할 수 있습니다. 놀라운 성능을 자랑하는 Chat GPT를 사용해보세요.

Sign To ChatGPT


사용자의 이미지를 통해 수어를 인식하고 Chat GPT에게 전달합니다. 수어를 통해 Chat GPT와 대화해보세요.

Open AI API


Open AI사는 다양한 인공지능 서비스를 제공하는 기업입니다. Open AI가 제공하는 다양한 API를 사용해보세요.
